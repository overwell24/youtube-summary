# youtube-summary
유튜브 동영상의 주요 내용을 자동으로 요약해 제공하는 서비스입니다. <br>
이 프로젝트는 캡스톤 디자인 수업에서 진행되었으며, 긴 영상의 핵심 내용을 빠르게 파악할 수 있도록 개발되었습니다.
